# Kodi Addon: File operations

## Description

This Kodi addon allows you to handle files directly from the context menu. It adds a menu item through which you can easily move,remove and rename files and directories.

**Addon ID:** script.fileops  
**Version:** 1.0.0  
**Author:** Heckie  
**License:** MIT  
**Platform:** All  
**Language:** English (en_GB)  

## Features

- Delete files from the context menu
- Rename files from the context menu
- Move files to other directories from the context menu
- Supports local paths (starting with /) and Windows paths (starting with drive letter)

## Installation

1. Download the addon's ZIP file.
2. Open Kodi and go to **Settings > Addons**.
3. Select **Install from ZIP file**.
4. Navigate to the downloaded ZIP file and install it.

Alternatively, you can install the addon from the official Kodi repository if available.

## Usage

1. Navigate to a file in Kodi (e.g., in a file manager or media library).
2. Open the context menu (usually with right-click or menu button).
3. Select the "Delete files" entry (Label 32002) to delete the file, "Rename file" (Label 32029) to rename it, or "Move file" (Label 32037) to move it to another directory.
4. For deletion: Confirm the action.
5. For renaming: Enter the new name and confirm.
6. For moving: Select the destination directory and confirm.

## Requirements

- Kodi with Python 3.0.0 or higher
- All platforms
